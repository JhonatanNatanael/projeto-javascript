Var

- Pode ser utilizada dentro de uma função, em programas que não suportam ES6.
Esta variável está em desuso depois da introdução de let e const.

Exemplos



Exemplo 1:
var nome= "Beatriz";
  console.log(nome);

Saída: Beatriz

Exemplo 2:
var idade= 25;
  console.log(idade);

Saída: 25

Let

- Let foi introduzido no JavaScript como uma alternativa para var. Pode ser utilizada
dentro de um bloco, com uma condição ou um loop. É recomendado
usar let para evitar erros no código.

Exemplos



Exemplo 1:
for (let i=0; i < 7; i++) {
  console.log(i);
}

Saída: 0, 1, 2, 3, 4, 5, 6

Exemplo 2:
let count= 20;
console.log(count);

count= 30;
console.log(count);
Saída: 20, 30

Const

- É usado para declarar uma variável que não pode ser declarada de outra forma
depois. Const pode ser colocada em um bloco no código.

Exemplos



Exemplo 1:
const arr1 = ["história", "matemática", "geografia", "biologia"];
  console.log(arr1.toString());
arr1[2] = "física";
  console.log(arr1.toString());

Saída: história, matemática, geografia, biologia
história, matemática, física, biologia

Exemplo 2:
const mult = 5;
function multiplyByConst(value) {
return value * mult;
}
  console.log(multiplyByConst(8));

Saída: 40
