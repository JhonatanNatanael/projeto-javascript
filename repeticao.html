While

- Executa um bloco de código enquanto uma condição for verdadeira.

Exemplos



Exemplo 1:
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Saída: 0, 1, 2, 3, 4

Exemplo 2:
let num = 10;
while (num > 0) {
  console.log(num);
num -= 2;
}

Saída: 10, 8, 6, 4, 2

For

- Executa um bloco de código um número específico de vezes de acordo com uma condição estabelecida.

Exemplos



Exemplo 1:
for (let i = 0; i < 5; i++) {
console.log(i);
}

Saída: 0, 1, 2, 3, 4

Exemplo 2:
const frutas = ["maçã", "banana", "uva"];
for (let i = 0; i < frutas.length; i++) {
console.log(frutas[i]);
}

Saída: "maçã", "banana", "uva"

.map

- Transforma cada elemento do array e retorna um novo array com os elementos transformados.

Exemplos



Exemplo 1:
const numeros = [1, 2, 3];
const dobrados = numeros.map(num => num * 2);
console.log(dobrados);

Saída: [2, 4, 6]

Exemplo 2:
const nomes = ["Ana", "Beto", "Carla"];
const cumprimentos = nomes.map(nome => `Olá, ${nome}!`);
console.log(cumprimentos);

Saída: ["Olá, Ana!", "Olá, Beto!", "Olá, Carla!"]

.filter

- Filtra elementos de um array de acordo com uma condição e retorna um novo array com os elementos que atendem à condição.

Exemplos



Exemplo 1:
const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(num => num % 2 === 0);
console.log(pares);

Saída: [2, 4]

Exemplo 2:
const palavras = ["árvore", "flor", "sol", "lua"];
const palavrasComO = palavras.filter(palavra => palavra.includes("o"));
console.log(palavrasComO);

Saída: ["árvore", "flor", "sol"]
