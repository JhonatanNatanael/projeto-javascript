JS objetos

- Os objetos são caixas de dados, que possuem propriedades e valores. Em que cada propriedade é um atributo do objeto, por exemplo: nome, idade e etc.
Já os valores são os conteúdos


Exemplo 1:
let maçã = { cor: "verde", sabor: "adocicado" };

JS objetos properties

-As propriedades são atributos relacionados ao objeto.

- As propriedades possuem nomes (strings), que servem para identificar os conjuntos de dados.

Para acessar as propriedades temos as seguintes sintaxes:



Exemplo 1:
console.log(maçã.cor);

Exemplo 2:
console.log(maçã["cor"]);

Relacionado a propriedade:


Para adicionar propriedade:
maçã.textura = "lisa";

Para alterar o valor:
maçã.cor = "azul";
