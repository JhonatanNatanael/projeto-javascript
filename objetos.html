JS Objects

- Os objetos são caixas de dados, que possuem propriedades e valores. Em que cada propriedade é um atributo do objeto, por exemplo: nome, idade e etc.
- Já os valores são os conteúdos.

Exemplos



Exemplo 1:
let maçã = { cor: "verde", sabor: "adocicado" }

Exemplo 2:
const pessoa = {
nome: "Ana",
idade: 25
};
console.log(pessoa.nome);
console.log(pessoa.idade);

Saída: Ana / 25

JS objects Properties

- As propriedades são atributos relacionados ao objeto.
- As propriedades possuem nomes (strings), que servem para identificar os conjuntos de dados.

Para acessar as propriedades temos as seguintes sintaxes:



Exemplo 1:
console.log(maçã.cor);

Exemplo 2:
console.log(maçã["cor"]);

Relacionado a propriedade:

Exemplos



Para adicionar propriedade:
maçã.textura = "lisa";

Para alterar o valor:
maçã.cor = "azul";

JS Object Methods

- É uma função que é definida dentro de um objeto.
- Os métodos são usados para realizar ações com os dados do objeto.

Exemplos



Exemplo 1:
const pessoa = {
nome: "Carlos",
saudacao: function() {
return Olá, meu nome é ${this.nome}!;
}
};
console.log(pessoa.saudacao());

Saída: Olá, meu nome é Carlos!

Exemplo 2:
const carro = {
marca: "Ford",
modelo: "Mustang",
detalhes: function() {
return Carro: ${this.marca} ${this.modelo};
}
};
console.log(carro.detalhes());

Saída: Carro: Ford Mustang

JS Object Display

- Exibe um objeto e mostra seu conteúdo.
- No JavaScript, a maneira mais comum de exibir um objeto é usando o console.log() ou convertendo-o para uma string JSON.

Exemplos



Exemplo 1:
const carro = {
marca: "Honda"
modelo: "Civic"
};
console.log(carro);
console.log(JSON.stringify(carro));

Saída: { marca: 'Honda', modelo: 'Civic' } / {"marca":"Honda","modelo":"Civic"}

Exemplo 2:
const produto = {
nome: "Notebook",
preco: 3000
};
console.log(produto);

Saída: { nome: 'Notebook', preco: 3000 }

JS Object Constructors

- É uma função usada para criar objetos.
- Construtores permitem criar múltiplos objetos semelhantes de forma eficiente.

Exemplos



Exemplo 1:
function Pessoa(nome, idade) {
this.nome = nome;
this.idade = idade;
}
const pessoa1 = new Pessoa("Maria", 28);
const pessoa2 = new Pessoa("João", 35);
console.log(pessoa1.nome);
console.log(pessoa2.idade);

Saída: Maria / 35

Exemplo 2:
function Animal(tipo, som) {
this.tipo = tipo;
this.som = som;
}
const cachorro = new Animal("Cachorro", "Latido");
const gato = new Animal("Gato", "Miau");
console.log(cachorro.som);
console.log(gato.tipo);

Saída: Latido / Gato
